                            QUICK START GUIDE
================================================================================

STEP 1: Start the Application
   $ docker-compose up -d

STEP 2: Create Admin User
   $ docker-compose exec web python manage.py createsuperuser

STEP 3: Access Application
   API:   http://localhost:8000
   Admin: http://localhost:8000/admin

STEP 4: Stop the Application
   $ docker-compose down

   DB_USER=postgres

docker-compose.yml (Development):

   - DB service: PostgreSQL 16 Alpine

================================================================================
                        TROUBLESHOOTING
================================================================================

Port Already in Use:
   1. Edit .env and change WEB_PORT
   2. Run: docker-compose down && docker-compose up -d

Database Connection Failed:
   1. Check status: docker-compose ps
   2. View logs: docker-compose logs db
   3. Restart: docker-compose restart db

Container Won't Start:
   1. View logs: docker-compose logs web
   2. Rebuild: docker-compose build --no-cache
   3. Start: docker-compose up -d

Permission Issues (Linux):
   1. Add user to docker group: sudo usermod -aG docker $USER
   2. Log out and log back in
